# Introduction 
This is a reimplitation of an older project of mine. The goal is to proccess User-Agents strings and related headers to determine what browser is being used and any details I could determine.
I have rewriten the code to use a SQLite Database and MS Entity Framework, to make things simplier to update and maintain.  This project makes heavy use of regular expressions.
